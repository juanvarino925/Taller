HTML

HTML (por sus siglas en inglés HyperText Markup Language, que significa Lenguaje de Marcado de Hipertexto) es el lenguaje estándar y fundamental utilizado para crear y estructurar el contenido de las páginas web. Fue inventado por Tim Berners-Lee en 1991 y se considera la columna vertebral de la World Wide Web.

CSS

CSS (por sus siglas en inglés Cascading Style Sheets, que significa Hojas de Estilo en Cascada) es el lenguaje utilizado para describir la presentación y el diseño visual de un documento estructurado, principalmente escrito en HTML. Es el responsable de la apariencia de una página web: colores, fuentes, tamaños, espaciado, diseños de múltiples columnas y animaciones.

JavaScript

JavaScript (JS) es un lenguaje de programación interpretado, principalmente utilizado para añadir interactividad, lógica y comportamiento dinámico a las páginas web. Es el tercer pilar fundamental de la web, junto con HTML (estructura) y CSS (estilo).
